# Electif_Blockchain_ZKN_11_2025

# Sujet: 
Application du protocole ZKP aux blockchaines.

# Contenu: 
Description du fonctionnement de ZKP dans sa globalité, avec des exemples simples. Approfondissement avec l'étude du protocole ZK-SNARK, utilisé dans les blockchains. Implementation d'une simulation de blockchaine, pour illustrer l'impact du cryptage ZK-SNARK sur l'opacité des transactions.
